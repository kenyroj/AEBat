如何加入

  • 將下列網址貼到瀏覽器上，下載該檔案並運行。
    https://tinyurl.com/ryb8q7e
  • 持續按「下一步」，中間不需要調整任何選項，直到安裝完成。
  • 打開命令提示字元(Win+R 輸入 cmd)會出現一個黑色視窗，並且顯示類似【C:\Users\你的使用者名字>】這樣的格式。
    C:\Users\User>
  • 於該視窗輸入下列指令
    git clone https://github.com/kenyroj/AnoEdnX.git
    建議用複製貼上的。(貼上方法為：滑鼠右鍵→貼上，不能按 Ctrl + V)
    此步驟是為了從網路上抓下所有相關檔案，並存放在【AnoEdnX】這個資料夾中。
  • 如下顯示【done.】後代表資料抓完，剩下的步驟請見【無限步驟】
    • Cloning into 'AnoEdnX'...
    remote: Enumerating objects: 75, done.
    remote: Counting objects: 100% (75/75), done.
    remote: Compressing objects: 100% (54/54), done.
    remote: Total 75 (delta 28), reused 64 (delta 17), pack-reused 0
    Unpacking objects: 100% (75/75), done.

工具更新

  • 如果日後相關檔案有發布更新，請參考以下步驟
  • 在時空貓資料夾輸入下列指令，會自動顯示一大串文字告訴你更新了什麼檔案。
    git pull
  • 若已經更新成最新版本，會顯示【Already up to date】。
    Already up to date.
