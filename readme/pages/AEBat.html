無限步驟

首次使用

  • 使用Windows電腦，下載【夜神模擬器】並安裝完成。
    https://tw.bignox.com/
    持續按「下一步」，中間不需要調整任何選項，直到安裝完成。
  • 打開命令提示字元【Win+R 輸入 cmd】會出現一個黑色視窗，輸入下列指令，移動到對應資料夾。
    CD AnoEdnX
  • 連接模擬器，輸入下列指令
    adb connect 127.0.0.1:62001
  • 檢查是否具有 root 權限，輸入下列指令
    adb shell
    會出現類似下列的提示符號
    root@shamu:/ #
    最後面是【#】則表示具有 Root 權限，按 Ctrl-D 可以退出回到命令提示字元

修改 ID

  • 修改 ID，請先輸入下列指令
    adb shell ls -l /data/data/games.wfs.anothereden/
    會出現類似下列的畫面，那個 u0_a43 就是你的 ID
    drwxrwx--x u0_a43 u0_a43 2020-03-02 23:50 app_textures
    drwx------ u0_a43 u0_a43 2020-03-08 02:01 app_webview
    輸入下列指令，以使用記事本來打開 AE.bat
    notepad AE.BAT
    會看到第一行就是【@SET UID=u0_a38】，請把它改成你剛剛看到的 u0_a43，然後存檔關閉。
    @SET UID=u0_a43

一般使用

  • 開啟模擬器、開啟時空貓
  • 打開命令提示字元【Win+R 輸入 cmd】，輸入下列指令
    CD Anoednx
    adb connect 127.0.0.1:62001
  • 自此可以開使用備份/還原大法。 輸入以下指令，把夜神裡的記錄備份到本機電腦裡
    AE S
    輸入以下指令，把本機電腦的記錄還原到夜神裡
    AE R
  • 進行動作之前先存檔
    AE S
  • 建議操作順序
    關閉網路
    選擇動作(ex:進幻璃、進入東方副本、強化武器)
    (不幸失敗)
    關閉時空貓
    AE R
    打開時空貓
    開啟網路

進階使用

  • 請參考 BAT工具 頁面，我們可以用指令來實現 打開/關閉網路 及 打開/關閉時空貓，那原本步驟的可以變成這樣
    WIFI 0
    選擇動作(ex:進幻璃、進入東方副本、強化武器)
    (不幸失敗)
    APP 0
    AE R
    APP 1
    WIFI 1
  • DOS 指令可以用 && 符號串在一起，所以可以一口氣下四個指令，我們的步驟變成這樣
    • WIFI 0
    選擇動作(ex:進幻璃、進入東方副本、強化武器)
    (不幸失敗)
    APP 0 && AE R && APP 1 && WIFI 1
  • 方向鍵【↑】和【↓】可以快速出現前面輸入過的指令，如此一來可以不用一直打重複的指令。
