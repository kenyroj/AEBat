無限引繼步驟 By 阿肯 (制服2020/03/12修訂)

【首次開啟需要經歷步驟】

1. 使用Windows電腦，上網搜尋【夜神模擬器】並下載完成。若為第一次下載，設備可能還沒ROOT，必須按上方設定(齒輪圖示)→基礎設定→啟動選項底下的【開啟Root】打勾→保存設定。

2. 按照LINE群組筆記本中的教學步驟，將Git工具安裝好、並抓好相關檔案。

3. 打開命令提示字元(Win+R 輸入 cmd)，會出現一個黑色視窗，輸入：【CD Anoednx(按enter)】，移動到對應資料夾。

4. 連接模擬器：輸入【adb connect 127.0.0.1:62001】並按enter
   (可直接複製貼上上條文字。貼上方法為：滑鼠右鍵→貼上，這裡不能按ctrl + v)
   若有顯示：【connected to 127.0.0.1:62001】即代表連接成功。

5. 檢查Root：輸入【adb shell】指令，若出現了【root@shamu:/ #】的指令，後面出現的是【#】，代表你有Root權限，才能繼續後續動作。
   按Ctrl-D 可以退出回到原資料夾。

6. 修改ID：輸入【adb shell ls -l /data/data/games.wfs.anothereden/】指令，會跑出好多行字，其中有幾行如下：
   drwxrwx--x u0_a43   u0_a43            2020-02-02 23:50 app_textures
   drwx------ u0_a43   u0_a43            2020-02-08 02:01 app_webview
   出現的u0_a43中的【43】是你的ID，要在其他地方修改這個數字。(每個人電腦都可能不一樣)
   先暫時不管黑色介面，改成從我的電腦沿著路徑打開位於【C:\Users\yao1993\AnoEdnX】的資料夾中，有個【AE.BAT】的檔案(就是肯大的程式)，滑鼠右鍵點選【編輯】。如果出現【Windows已保護你的電腦】之類的警示，請點選【其他資訊】→【仍要執行】
   成功打開AE.BAT後，最上面那行的【@SET UID=u0_a38】，請把【38】改成你剛剛看到的ID(例如：43)，然後儲存並離開。

7. 恭喜，你終於完成了前置設定。之後使用可直接參考【之後每次要使用的步驟】。


【之後每次要使用的步驟】

1. 開啟模擬器、開啟時空貓

2. Win+R 輸入 cmd → CD Anoednx → adb connect 127.0.0.1:62001

3. 開始使用備份/引繼大法。
   輸入 AE S 代表你要把時空貓的存檔備份到PC裡。
   輸入 AE R 代表你要把剛剛備份的存檔放回時空貓儲存空間中。

建議操作順序：AE S → 關網路 → 選擇動作(ex:進幻璃、進入東方副本、強化武器) → (不幸失敗) → 關閉時空貓 → AE R → 開啟網路、時空貓 → 關網路 → 選選擇動作 (以上重複) )=

關閉網路方法：夜神模擬器的桌面→ Tools 資料夾 → 設定 → Wifi。 可善用右下角的兩個長方形相疊的那個鍵快速切換時空貓/wifi設定，以求快速。
或是參考【2.Utility.txt】中所提供的指令，直接輸入【WIFI 0】及【WIFI 1】開關WIFI。

關閉遊戲的方法：
關閉網路方法：右下角的兩個長方形相疊的那個鍵→用滑鼠把APP往左或往右拖曳，如手機滑掉一樣→完成關閉
或是參考【2.Utility.txt】中所提供的指令，直接輸入【APP 0】及【APP 1】開關遊戲。


【進階作法】

1. doskey的使用方法
輸入 【doskey 甲 = 乙】，即可以輸入甲的方式取代乙。
舉例：輸入 doskey R = AE R，則之後只要輸入R，等同於輸入AE R指令產生的效果。

*每次重開後必須重新設定doskey。
*doskey目前無法判定"&&"符號後的指令，故尚不能多串指令並用。

2. &&的使用方法
輸入&&連接兩指令的話，會按照順序將指令跑完，不用再分批輸入。
舉例：輸入 WIFI 1 && AE S ，會先打開WIFI，然後再存檔備份。

3. 方向鍵的用法
按方向鍵【↑】和【↓】可以快速出現前面輸入過的指令，如此一來可以不用一直打重複的指令。


4. 目前推薦的「合體技」

步驟一：做行動前，先輸入一次 AE S。
步驟二：輸入【WIFI 0 】(關閉網路)
步驟三：進行動作，若不幸失敗，則接續步驟四。
步驟四：輸入【 APP 0 && AE R && WIFI 1 && APP 1 】 (關閉遊戲→洗掉紀錄→開啟網路→重開遊戲 一步到位)
步驟五：按繼續遊戲。
步驟六：回到步驟二，重複步驟二~步驟六。記得可以用方向鍵【↑】來省下輸入的時間。
