無限引繼步驟 By 阿肯 (制服2.25修訂)

【首次開啟需要經歷步驟】

1. 使用Windows電腦，上網搜尋【夜神模擬器】並下載完成。若為第一次下載，設備可能還沒ROOT，必須按上方設定(齒輪圖示)→基礎設定→啟動選項底下的【開啟Root】打勾→保存設定

2. 下載ADB工具及阿肯程式(都放在同個資料夾中)：https://pse.is/QLVUS 下載完成後建議把這個資料夾放在桌面上較方便。
 
3. 打開命令提示字元(Win+R 輸入 cmd)，會出現一個黑色視窗。此時先透過輸入切換目錄的方式(輸入格式：CD 路徑名稱 中間空一格)到剛剛資料夾的目錄。
   如果剛剛下載的檔案確實放在桌面上，輸入順序為：CD desktop (按enter) → CD PT (按enter)

4. 連接模擬器：輸入【adb connect 127.0.0.1:62001】並按enter
   (可直接複製貼上上條文字。貼上方法為：滑鼠右鍵→貼上，這裡不能按ctrl + v)
   若有顯示：【connected to 127.0.0.1:62001】即代表連接成功。

5. 檢查Root：輸入【adb shell】指令，若出現了【root@shamu:/ #】的指令，後面出現的是【#】，代表你有Root權限，才能繼續後續動作。
   按Ctrl-D 可以退出回到原資料夾。

6. 修改ID：輸入【adb shell ls -l /data/data/games.wfs.anothereden/】指令，會跑出好多行字，其中有幾行如下：
   drwxrwx--x u0_a43   u0_a43            2020-02-02 23:50 app_textures
   drwx------ u0_a43   u0_a43            2020-02-08 02:01 app_webview
   出現的u0_a43中的【43】是你的ID，要在其他地方修改這個數字。(每個人電腦都可能不一樣)
   先暫時不管黑色介面，改成打開桌面上的PT資料夾中，有個【AE.BAT】的檔案(就是肯大的程式)，滑鼠右鍵點選【編輯】。如果出現【Windows已保護你的電腦】之類的警示，請點選【其他資訊】→【仍要執行】
   成功打開AE.BAT後，最上面那行的【@SET UID=u0_a36】，請把【36】改成你剛剛看到的ID(例如：43)，然後儲存並離開。

7. 恭喜，你終於完成了前置設定。之後使用可直接參考【之後每次要使用的步驟】。


【之後每次要使用的步驟】

1. 開啟模擬器、開啟時空貓

2. Win+R 輸入 cmd → CD desktop  → CD PT → adb connect 127.0.0.1:62001

3. 開始使用備份/引繼大法。
   輸入 AE S 代表你要把時空貓的存檔備份到PC裡。
   輸入 AE R 代表你要把剛剛備份的存檔放回時空貓儲存空間中。

(建議操作順序：AE S → 關網路 → 選擇動作(ex:幻璃選路、進入東方副本、強化武器) → (不幸失敗) → 關閉時空貓 → AE R → 開啟網路、時空貓 → 關網路 → 選選擇動作 (以上重複)
建議關閉網路方法：夜神模擬器的桌面→ Tools 資料夾 → 設定 → Wifi。 可善用右下角的兩個長方形相疊的那個鍵快速切換時空貓/wifi設定，以求快速。


*修改歷程註明
1) 原本AE.bat和ADB工具要分開來下載，為了方便後人取用，我打包成一個檔案上傳到雲端簡化了步驟
2) 原本AE.bat中，備份是"B"、回溯是"R"，但因為B和R用某些電腦的字體看易混淆，故我擅自修改了.bat檔，將備份改成了代號"S"
